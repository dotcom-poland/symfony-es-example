## Running locally

1. Install [Docker](https://docs.docker.com/engine/install/)
2. Copy `docker-compose.custom.yml.dist` to `docker-compose.custom.yml` and modify if necessary
3. `docker compose up -d`

## Stopping

`docker compose down`
